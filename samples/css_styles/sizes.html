Margins

- {{mValue}}

Для получения отступа только слева добавьте к названию класса l. Например lm24 - отступ слева 24px.
Для получения отступа только справа добавьте к названию класса r.
Для получения отступа только вверху добавьте к названию класса t.
Для получения отступа только внизу добавьте к названию класса b.

Paddings

Для получения внутреннего отступа вместо буквы m используйте p. Например p24. Размеры отступов такие же как и внешние.
